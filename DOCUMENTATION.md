## Documentation

You can see below the API reference of this module.

### `decode(input)`
Decodes an encoded string.

#### Params
- **String** `input`: The encoded string.

#### Return
- **String** The decoded string.

### `encode({})`
Encodes a string.

#### Params
- **** `{}`: input The string that must be encoded.

#### Return
- **String** The encoded string.
